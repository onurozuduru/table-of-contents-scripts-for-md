# Create Table of Contents for Markdown
---------------------------------------

Scripts that creates in-line links for titles of github flavored markdown.

I had free couple hours so, I challenged myself to write awk one liner which 
finds headlines in a Readme file and gives list of headlines as in-line link 
format for github flavored markdown :poop: 

It didn't take too much time then, I decided to create a bash script. 
It was also not time consuming so, I created a Python script that does the 
same job without using regex.

At the end, I wasted my short free time but, it was fun. And I'm putting this 
to Github because, why not :smile:

## Usage

### Bash Script

```bash
  $ ./table_of_contents.sh <filename>
```

### Python Script

```bash
  $ ./table_of_contents.py -i <inputfile> -o <outputfile>
```
